# Mic-Me
### Description: Mictalk is a simple sweb application that allows a user to have a virtual microphone and search lyrics for several songs.
>
### Installation & Setup
>1. Clone the project
>2. cd mic-talk
>3. npm i
>4. npm run dev
>
